What is Python?
Python is a high-level, easy-to-learn programming language used for:
	• Web development
	• Data Science
	• Machine Learning
	• Artificial Intelligence
	• Automation
	• App development
It is one of the most popular programming languages in the world.

Simple Definition
Python is a programming language used to give instructions to computers.
